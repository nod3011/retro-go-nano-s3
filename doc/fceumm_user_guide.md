# คู่มือการใช้งาน FCEUMM (NES Emulator)

คู่มือนี้จะช่วยให้คุณใช้งาน Emulator NES (FCEUMM) บนเครื่อง Retro-Go ได้อย่างเต็มประสิทธิภาพ ทั้งการควบคุมพื้นฐาน ปุ่มลัด (Combo) และการเปลี่ยนแผ่นสำหรับเกม Famicom Disk System (FDS)

---

## 🎮 การควบคุมพื้นฐาน (Basic Controls)

- **D-Pad (ขึ้น/ลง/ซ้าย/ขวา):** บังคับทิศทาง
- **ปุ่ม A:** ปุ่ม A ของ NES
- **ปุ่ม B:** ปุ่ม B ของ NES
- **ปุ่ม START:** เริ่มเกม / Pause
- **ปุ่ม SELECT:** เลือกโหมดในเกม

---

## 🔥 ปุ่มลัดพิเศษ (Button Combos)

เครื่อง Retro-Go มีปุ่ม **MENU** เป็นปุ่มหลักในการใช้งานฟังก์ชันเสริม โดยการ **กดปุ่ม MENU ค้างไว้** แล้วกดปุ่มอื่นดังนี้:

- **MENU + A:** เปิด/ปิด **Turbo A** (รัวปุ่ม A อัตโนมัติ)
- **MENU + B:** เปิด/ปิด **Turbo B** (รัวปุ่ม B อัตโนมัติ)
- **MENU (กดแล้วปล่อย):** เปิด **Game Menu** (สำหรับ Save/Load State)
- **ปุ่ม OPTION:** เปิด **Options Menu** (สำหรับจัดการ Cheat Code)

---

## 💾 การใช้งาน Famicom Disk System (FDS)

สำหรับเกมที่ต้องมีการเปลี่ยนแผ่นหรือกลับด้านแผ่น (เช่น Zelda 1 หรือ Metroid ภาคแรก) ให้ใช้คำสั่ง Combo ดังนี้:

1. **กด MENU + DOWN:** เพื่อ **Eject Disk** (เอาแผ่นออก)
2. **กด MENU + SELECT:** เพื่อ **Switch Side** (เปลี่ยนด้านแผ่น เช่น จาก Side A เป็น Side B)
3. **กด MENU + UP:** เพื่อ **Insert Disk** (ใส่แผ่นกลับเข้าไป)

> [!TIP]
> เมื่อเกมขึ้นหน้าจอให้ "Please set Side B" ให้คุณกด **MENU + SELECT** ตามด้วย **MENU + UP** เพื่อให้ emulator รับรู้การเปลี่ยนด้านแผ่นครับ

---

## 🛠️ การจัดการสูตรโกง (Cheats)

1. กดปุ่ม **OPTION** ระหว่างเล่นเกม
2. เลือกเมนู **Cheats**
3. คุณสามารถกดปุ่ม **A** เพื่อเปิด/ปิดสูตรที่มีอยู่ หรือเลือก **Add new cheat...** เพื่อใส่ Code ใหม่ (รองรับ Game Genie และ PAR)
