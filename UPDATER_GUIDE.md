# คู่มือการใช้งาน Self-Updater (NANO-S3)

การทำงานของระบบ Self-Updater จะช่วยให้คุณสามารถอัปเดต Firmware ได้โดยตรงจาก SD Card โดยไม่ต้องต่อ USB หลังจากการติดตั้งครั้งแรก

## 1. การเตรียมเครื่องครั้งแรก (First-time Setup)
เนื่องจากมีการเพิ่ม Partition `updater` ใหม่:
- รันคำสั่ง: `python rg_tool.py build-img --target nano-s3`
- Flash ไฟล์ `.img` ที่ได้ลงเครื่องผ่าน USB เพื่อสร้างโครงสร้าง Partition ล่าสุด

## 2. การสร้างไฟล์สำหรับอัปเดต (.img)
เมื่อมีการแก้ไขโค้ดและต้องการส่งให้ผู้ใช้อื่นอัปเดต:
- รันคำสั่ง Build เพื่อสร้างไฟล์ Image ตามปกติ
- ไฟล์จะอยู่ที่: `build/nano-s3/retro-go.img`
- แนะนำให้เปลี่ยนชื่อไฟล์เพื่อป้องกันความสับสน เช่น `retro-go-nano-s3-v1.1.img`

## 3. ขั้นตอนการอัปเดตผ่านตัวเครื่อง (Update Flow)
1. **เตรียม SD Card**:
   - ก๊อปปี้ไฟล์ `.img` ไปไว้ที่ `/sd/nano-s3/firmware/`
2. **สั่งอัปเดตจากเครื่อง**:
   - เปิดเครื่องไปที่เมนู **About (เกี่ยวกับ)** -> **Check for updates (ตรวจสอบการอัปเดต)**
   - Launcher จะค้นหาไฟล์ใน Folder สั่ง Reboot เข้าแอป `updater` โดยอัตโนมัติ
3. **ขั้นตอนการ Flash**:
   - แอป `updater` จะทำงานและตรวจสอบไฟล์ที่เลือก
   - ระบบจะทำการเขียนข้อมูลลง Partition ต่างๆ ตามชื่อ (เช่น `launcher`, `retro-core`)
   - **ความปลอดภัย**: ระบบจะข้าม Partition สำคัญ (เช่น `nvs`, `phy_init`) และข้ามตัวมันเอง (`updater`) เสมอ
   - เมื่อเสร็จสิ้น เครื่องจะ Reboot กลับเข้าสู่ Launcher ตัวใหม่

## ข้อสังเกต
- ตัว Updater จะทำการอัปเดตเฉพาะ Partition ที่พบในไฟล์ `.img` เท่านั้น
- หากต้องการอัปเดตเฉพาะแอปบางตัว สามารถสร้าง Image ขนาดเล็กที่มีแค่ Partition นั้นๆ ได้
